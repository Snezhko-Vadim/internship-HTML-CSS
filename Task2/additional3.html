Тег <figure> и <figcaption>

Описание тегов

<figure>
Данный тег является блочным. Предназначен для отображения картинок , диаграмм, видео, аудио, кода
с необязательной подписью. Используется в том случае , если содержимое относится к
отдельному элементу кода и может быть удалено из потока без потери смысловой нагрузки контента
<figcaption>
Данный тег является описанием для тега <figure> и используется вместе с ним.
Должен быть единственным в теге <figure>. Вставляется или перед контентом,
или после контента тега <figure>.

Плюсы и минусы

Плюсы:

  • Не надо создавать блок для картинки и подписи вмесе.
    Получается как бы инкапсуляция => лучшая читаемость кода
  • Можно группировать несколько изображений
  • Могут быть проблемы с отображением в некоторых
    браузерах, т.к. тег моложе чем img

Минусы:

  • По смыслу используется только если помещаемый контент
    будет связан с каким-то элементом страницы. Т.е. при
    его удалении смысл страницы не потеряется.
  • С помощью него не удобно будет загружать фоновые изображения
