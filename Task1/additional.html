Малоиспользуемые теги HTML

<pre>
С помощью данного тега можно отображать текст в том виде , в каком мы его напечатаем: 
любое    кол-во        пробелов
                    и
                любые
            переносы
<small>
можем уменьшать текст вот так :
text
и даже так:
text
<tt>
этот тег делает буквы в словах одинаковых размеров с одинаковыми кегельными площадками:типа вот так
<code>
служит для выделения програмного кода в тексте: let hello = (name) => alert(`${name},hello`)
<sup>,<sub>
sup и sub используется для формул как верхний и нижний индекс: a2 an
<kbd>
служит для отображения названий клавиш: Alt Shift
<big>
можем увеличивать текст вот так :
text
и даже так:
text
<blockquote>
тег для выделения больших цитат:
Здесь какая-то большая цитата
<q>
для маленьких цитат: маленькая цитатка
<ins>
как я понял это тег для обновлённой информации. Выглядит так: Обновочка
<abbr>
тег для аббривиатур, типа: JS
<nobr>
для отображения текста без переносов одной строкой
<adress>
для хранения информации об авторе:
инфа про меня
<wbr>
тег указывает на место, в котором будет перенос текста при уменьшении родительского элемента:
это предложение должно стать вертикальным
