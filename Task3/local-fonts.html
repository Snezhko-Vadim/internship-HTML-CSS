Google Fonts Collection

The easiest way to use is Google Fonts. There it is enough to choose a suitable font from a rich collection, customize it (select the desired styles and languages) and get a ready-made reference.

Self-connection

Not all fonts can be found on Google Fonts, plus sometimes you want the loading of resources to be independent of a third-party service, even if it is as reliable as Google. Finally, a layout designer should be able to connect fonts.

The main problem when working with fonts is the variety of formats. If you download any font from the same Google Fonts, you will get a chunky zip file containing usually three (or even more) different files for the same font. Each style will need to be specified with a separate rule. If you want to include regular, bold, italic, and bold italic fonts, you have to write four instructions.
